1. Рекомендации для настроек продукта

    По умолчанию в используется интегрированная с Apache база данных Derby. Для данного продукта Вы так же можите использовать MySQL, Postgres или например IBM DB2 или Oracle

  2. Для возможности загрузки и импорта изображений на доску

    • Установите ImageMagick на Ваш сервер, более подробную информацию касающуюся установки можно посмотреть тут http://www.imagemagick.org Инструкцию по установке Вы можите найте здесь http://www.imagemagick.org/script/binary-releases.php, однако в большинстве Линукс систем Вы можите установить это через Ваш менеджер пакетов (apt-get it)

  3. Для возможности импорта PDF файлов на доску

    • Установите GhostScript на ваш сервер, более подробную ниформацию вы можите посмотреть на сайте http://pages.cs.wisc.edu/~ghost/ Инструкции по установке Вы можите найти здесь, однако в большинстве Линукс систем Вы можите установить это через Ваш менеджер пакетов (apt-get it).
    • Установите SWFTools на Ваш, более подробную информацию можно найти на сайте http://www.swftools.org/ Некоторые Линукс дистрибутивы уже включили его в свои менеджеры пакетов, посмотреть можно здесь http://packages.debian.org/unstable/utils/swftools), для установки рекомендуется SWFTools версии 0.9 так как предыдущая версия содержит баг который приводил к ошибке отображения размеров объкета на доске

  4. Для возможности импорта файлов .doc, .docx, .ppt, .pptx, ... всех документов на доску

    • OpenOffice-Service запускается и слушает порт 8100, подробности можно посмотреть здесь OpenOfficeConverter

  5. Для возможности Записи и импорта файлоф .avi, .flv, .mov и .mp4 на доску

    • Установите FFMpeg. Вы должны взять последнюю версию FFMPEG! Для Windows Вы можите скачать программу с этого сайта http://ffmpeg.arrozcru.org/builds/ Пользователи Linux или OSx должен быть в состоянии использовать одну из различных Инсталляционных Инструкций в Сети. Вы должны установить libmp3lame!
    • Установите SoX http://sox.sourceforge.net/. Вы должны установить последнюю версию SoX! SoX 12.xx не будет работать!
